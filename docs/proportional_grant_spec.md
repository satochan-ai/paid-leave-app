# 比例付与仕様書

## 目的

週所定労働日数が少ない労働者（パート・時短勤務等）について、労働基準法に基づく比例付与日数を計算する。

---

## 通常付与対象

以下のいずれかに該当する場合は通常付与として扱う。

| 条件 | 内容 |
|------|------|
| workType が normal | 明示的に通常付与に設定されている |
| 週所定労働時間 ≥ 30時間 | 時間で判定する場合 |
| 週所定労働日数 ≥ 5日 | 日数で判定する場合 |
| 年間所定労働日数 ≥ 217日 | 年間日数で判定する場合 |

---

## 比例付与対象

以下をすべて満たす場合は比例付与となる。

- 週所定労働時間 < 30時間
- 週所定労働日数 < 5日（4日以下）

または、年間所定労働日数が 48〜216日の範囲にある場合。

---

## 通常付与テーブル

| 付与回次 | 継続勤務年数 | 付与日数 |
|---------|------------|---------|
| 1回目 | 0.5年 | 10日 |
| 2回目 | 1.5年 | 11日 |
| 3回目 | 2.5年 | 12日 |
| 4回目 | 3.5年 | 14日 |
| 5回目 | 4.5年 | 16日 |
| 6回目 | 5.5年 | 18日 |
| 7回目以降 | 6.5年以上 | 20日 |

---

## 比例付与テーブル

| 週所定労働日数 | 0.5年 | 1.5年 | 2.5年 | 3.5年 | 4.5年 | 5.5年 | 6.5年以上 |
|---:|---:|---:|---:|---:|---:|---:|---:|
| 4日 | 7 | 8 | 9 | 10 | 12 | 13 | 15 |
| 3日 | 5 | 6 | 6 | 8 | 9 | 10 | 11 |
| 2日 | 3 | 4 | 4 | 5 | 6 | 6 | 7 |
| 1日 | 1 | 2 | 2 | 2 | 3 | 3 | 3 |

---

## 年間所定労働日数による判定

週所定労働日数ではなく年間所定労働日数で管理する場合は、以下の換算表を使用する。  
年間所定労働日数が比例付与範囲（48〜216日）にある場合、週日数判定より優先される。

| 年間所定労働日数 | 週日数相当 |
|---:|---:|
| 217日以上 | 通常付与 |
| 169〜216日 | 4日 |
| 121〜168日 | 3日 |
| 73〜120日 | 2日 |
| 48〜72日 | 1日 |

---

## 勤務条件データ項目

| 項目 | フィールド名 | 型 | デフォルト |
|------|------------|-----|----------|
| 付与区分 | workType | "normal" \| "proportional" \| "custom" | "normal" |
| 週所定労働日数 | weeklyWorkDays | number | 5 |
| 週所定労働時間 | weeklyWorkHours | number | 40 |
| 年間所定労働日数 | annualWorkDays | number | 260 |

既存社員で勤務条件が未設定の場合はすべてデフォルト値で補完し、通常付与として扱う。

---

## 動作確認例

```js
// 通常付与
calculateGrantDays("2024-04-01", "2024-10-01", {
  workType: "normal", weeklyWorkDays: 5, weeklyWorkHours: 40, annualWorkDays: 260
})
// → 10

// 比例付与 週3日
calculateGrantDays("2024-04-01", "2024-10-01", {
  workType: "proportional", weeklyWorkDays: 3, weeklyWorkHours: 24, annualWorkDays: 156
})
// → 5

calculateGrantDays("2024-04-01", "2025-10-01", {
  workType: "proportional", weeklyWorkDays: 3, weeklyWorkHours: 24, annualWorkDays: 156
})
// → 6

// 比例付与 週4日
calculateGrantDays("2024-04-01", "2024-10-01", {
  workType: "proportional", weeklyWorkDays: 4, weeklyWorkHours: 28, annualWorkDays: 208
})
// → 7

// 週30時間以上 → 通常付与
calculateGrantDays("2024-04-01", "2024-10-01", {
  workType: "proportional", weeklyWorkDays: 4, weeklyWorkHours: 30, annualWorkDays: 208
})
// → 10

// 年間217日以上 → 通常付与
calculateGrantDays("2024-04-01", "2024-10-01", {
  workType: "proportional", weeklyWorkDays: 4, weeklyWorkHours: 28, annualWorkDays: 217
})
// → 10
```
