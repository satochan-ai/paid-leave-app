1. デモデータ初期化（ダッシュボード → ⚠ 危険操作 → デモデータ初期化）
2. ブラウザのズームを 100% に設定
3. ウィンドウ幅を 1280px 以上に設定
4. 不要なブラウザ通知・拡張機能を非表示にする
