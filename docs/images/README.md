## 撮影ルール

- ブラウザ幅はPC表示（1280px以上）を基本にする
- 個人情報はデモデータのみ使用する
- 実在の社員名・メールアドレスは使わない
- 不要なブラウザタブや通知は映さない
- 画像はPNG形式を基本にする
