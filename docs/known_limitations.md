# 既知の制限事項

このアプリはlocalStorageを使ったプロトタイプです。以下の制限があります。

---

## データ保存・共有

| 制限 | 詳細 |
|------|------|
| 端末・ブラウザ依存 | localStorageはブラウザ・端末ごとに独立しているため、別端末・別ブラウザからはデータを参照できない |
| データ消失リスク | ブラウザのキャッシュ削除・localStorageクリアでデータが消える |
| 複数管理者の同時利用不可 | データ競合の仕組みがなく、複数端末から同時操作した場合の整合性は保証しない |
| サーバーDB保存なし | データはすべてクライアント（ブラウザ）側にのみ存在する |

---

## 認証・セキュリティ

| 制限 | 詳細 |
|------|------|
| パスワード平文保存 | パスワードはlocalStorageに平文で保存されており、開発者ツールで確認できる |
| パスワード平文表示 | 管理者画面の社員詳細に仮パスワードが平文表示される（プロトタイプ用仕様） |
| クライアントサイド認証のみ | 開発者ツールでlocalStorageを直接操作するとアクセス制御を回避できる |
| セッション管理なし | タブを閉じてもlocalStorageにセッション情報が残る |

---

## 業務ロジック

| 制限 | 詳細 |
|------|------|
| 出勤率8割判定未対応 | 労働基準法上の出勤率要件の判定は行っていない |
| 休職期間を考慮した厳密計算未対応 | 休職期間が出勤率・付与日数に影響するケースは未対応 |
| 有給申請承認フロー未対応 | 社員からの申請と管理者の承認フローはない |
| メール通知未対応 | 失効予定アラートなどのメール通知機能はない |
| 時間単位有給未対応 | 時間単位の有給取得には対応していない（日単位・半日単位のみ） |
| CSVインポート未対応 | CSVからのデータ取り込みはできない |

---

## UI・操作

| 制限 | 詳細 |
|------|------|
| 操作履歴（Undo）なし | 削除・変更操作を元に戻す機能はない |
| 印刷・PDF出力未対応 | 画面の帳票出力は未対応 |
| Excel xlsx出力未対応 | CSV出力のみ対応（xlsxはなし） |
| 自動バックアップなし | 定期的な自動バックアップは未対応 |
