# CSV出力仕様書

## 対応CSV

| CSV名 | ファイル名形式 | 出力画面 |
|-------|-------------|---------|
| 社員一覧CSV | `employees_YYYYMMDD_HHMMSS.csv` | ダッシュボード・社員一覧 |
| 有給残日数一覧CSV | `leave_summary_YYYYMMDD_HHMMSS.csv` | ダッシュボード・社員一覧 |
| 有給取得履歴CSV | `leave_usages_YYYYMMDD_HHMMSS.csv` | ダッシュボード |
| 社員別詳細CSV | `employee_detail_社員名_YYYYMMDD_HHMMSS.csv` | 社員詳細 |

## 文字コード

BOM付きUTF-8（`﻿` プレフィックス付き Blob）  
理由：Excelで開いたときの日本語文字化けを防ぐため

## 改行コード

CRLF（`\r\n`）

## エスケープ仕様

| 条件 | 処理 |
|------|------|
| `null` / `undefined` | 空文字に変換 |
| ダブルクォート (`"`) を含む | `""` に変換し、全体をダブルクォートで囲む |
| カンマ (`,`) を含む | 全体をダブルクォートで囲む |
| 改行 (`\n` / `\r`) を含む | 全体をダブルクォートで囲む |

## 権限

管理者のみ出力可能（各関数冒頭で `isAdmin()` を確認）  
社員ログイン時はCSVボタンが表示されず、直接関数を呼んでもアラートを表示して処理を停止する

## 社員一覧CSV ヘッダー

```
社員ID, 社員名, メールアドレス, 入社日, 退職日, ステータス, 備考, 作成日時, 更新日時
```

- ステータスは日本語表示（`active` → 在籍 / `leave` → 休職中 / `retired` → 退職）
- 退職者も出力対象

## 有給残日数一覧CSV ヘッダー

```
社員ID, 社員名, メールアドレス, ステータス, 入社日, 総付与日数, 総取得日数, 有効残日数, 失効済み残日数, 次回付与日, 次回付与日数, 直近失効日, 直近失効日数
```

- `calculateLeaveSummary()` を使って計算
- 日付なし項目は空文字

## 有給取得履歴CSV ヘッダー

```
取得ID, 社員ID, 社員名, メールアドレス, 取得日, 取得日数, 備考, 登録者ID, 登録日時
```

- 取得日降順 → 社員名昇順でソート

## 社員別詳細CSV 構成

1ファイル内に複数セクションを格納する形式。

```
セクション, 項目, 値
基本情報, 社員ID, ...
基本情報, 社員名, ...
...

有給サマリー, 総付与日数, ...
...

付与履歴, 付与日, 付与日数, 使用済み日数, 残日数, 失効日, 状態
付与履歴, 2024-10-01, 10, 2, 8, 2026-10-01, 未失効
...

取得履歴, 取得日, 取得日数, 備考, 登録日時
取得履歴, 2025-12-10, 1, 私用, ...
```

- セクション間は空行で区切る
- ファイル名の社員名は OS予約文字（`\ / : * ? " < > |`）を `_` に置換
