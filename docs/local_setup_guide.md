# ローカル起動ガイド

## 推奨起動方法

このアプリはHTML/CSS/JavaScriptのみで構成されていますが、`file://` で直接開くのではなく、ローカルサーバー経由で起動してください。

---

## Pythonを使う場合

```bash
cd C:\Users\satoc\work\paid-leave-app
python -m http.server 8080
```

ブラウザで以下にアクセスします。

```
http://localhost:8080
```

---

## VS Code Live Serverを使う場合

1. VS Codeでプロジェクトフォルダを開く
2. Live Server拡張機能をインストール（`ritwickdey.LiveServer`）
3. `index.html` を右クリック
4. 「Open with Live Server」を選択

ブラウザが自動で開き、ファイル変更時にリアルタイム更新されます。

---

## Node.js を使う場合

```bash
npx serve .
```

または

```bash
npx http-server . -p 8080
```

---

## `file://` で開く場合の注意

`file://` で直接開くと、ブラウザのセキュリティ制限により以下が発生する場合があります。

- ログインができない
- 画面遷移しない
- データが保存されない

**ログインできない場合は、必ずローカルサーバーまたはLive Serverで起動してください。**

## スクリーンショット撮影を行う場合

このプロジェクトでは、Playwright を使ったスクリーンショット自動撮影スクリプトを用意しています。

### 依存関係のインストール

```bash
npm install
```

### 撮影実行

```bash
npm run screenshots
```

スクリプトが内蔵サーバーを自動起動するため、事前にサーバーを立ち上げる必要はありません。  
撮影結果は `docs/images/` に保存されます。

---

## デモデータについて

初回起動時、またはデモデータ初期化後に、以下のサンプルデータが自動投入されます。

- 管理者アカウント（1名）
- 社員アカウント（山田 太郎）
- 有給付与履歴（2件）
- 有給取得履歴（2件）

データはブラウザのlocalStorageに保存されます。
ブラウザのキャッシュをクリアするとデータが消えますので、事前にJSONバックアップを推奨します。
