## 1. ログイン画面

| 項目 | 内容 |
|------|------|
| 画面目的 | メールアドレスとパスワードでログインする |
| URL | index.html |
| 権限 | 誰でもアクセス可能（ログイン済みはリダイレクト） |
| 遷移先 | admin → dashboard.html / employee → mypage.html |

### 表示項目
- アプリタイトル

### 入力項目
| フィールド | タイプ | 必須 |
|-----------|--------|------|
| メールアドレス | email | ○ |
| パスワード | password | ○ |

### ボタン
- ログインボタン（submit）

### エラー表示エリア
- メールアドレス未入力
- パスワード未入力
- メールアドレスまたはパスワードが一致しない

### バリデーション
- メールアドレス・パスワードの未入力チェック
- 認証失敗時のエラーメッセージ表示

## 4. 社員追加・編集画面

| 項目 | 内容 |
|------|------|
| 画面目的 | 社員情報を新規登録または編集する |
| URL（新規） | pages/admin/employee-form.html |
| URL（編集） | pages/admin/employee-form.html?id=emp_xxx |
| 権限 | admin のみ |
| 遷移先 | 保存後 → 社員一覧画面 |

### 入力項目
| フィールド | タイプ | 必須 | 備考 |
|-----------|--------|------|------|
| 社員名 | text | ○ | |
| メールアドレス | email | ○ | 重複チェックあり |
| パスワード | password | ○（新規のみ） | 編集時は空欄で変更しない |
| 権限 | select | ○ | admin / employee |
| 入社日 | date | ○ | |
| 退職日 | date | - | 退職時のみ入力 |
| ステータス | select | ○ | active / leave / retired |
| 備考 | textarea | - | |

### ボタン
- 登録するボタン（submit）
- キャンセルボタン（社員一覧へ戻る）

### バリデーション
- 社員名未入力
- メールアドレス未入力・形式不正
- メールアドレス重複
- パスワード未入力（新規のみ）
- 入社日未入力
- 退職日が入社日より前

## 6. 有給取得登録画面

| 項目 | 内容 |
|------|------|
| 画面目的 | 社員の有給取得を登録する |
| URL | pages/admin/leave-usage-form.html?employeeId=emp_xxx |
| 権限 | admin のみ |
| 遷移先 | 登録後 → 社員詳細画面 |

### 表示項目
- 対象社員名（読み取り専用）
- 現在の有給残日数（アラート形式で表示）

### 入力項目
| フィールド | タイプ | 必須 | 備考 |
|-----------|--------|------|------|
| 取得日 | date | ○ | |
| 取得日数 | select | ○ | 0.5 / 1 / 2 / 3 / 4 / 5 |
| 備考 | textarea | - | |

### ボタン
- 登録するボタン（submit）
- キャンセルボタン（社員詳細へ戻る）

### バリデーション・登録時処理
- 取得日未入力
- 取得日数未入力・0以下
- 残日数を超える取得
- 退職済み社員への登録
- 登録成功後：古い付与分から順に消化して保存

## 7. 社員マイページ

| 項目 | 内容 |
|------|------|
| 画面目的 | ログイン中の社員が自分の有給情報を確認する |
| URL | pages/employee/mypage.html |
| 権限 | employee（自分のデータのみ） |
| 遷移先 | なし（閲覧専用） |

### 表示項目（プロフィールヘッダー）
- 氏名 / 入社日 / 現在の有給残日数（大きく表示）

### 表示項目（有給サマリー）
- 今年付与された日数
- 今年取得した日数
- 次回付与日
- 次回付与予定日数
- 失効予定日数（30日以内）

### 表示項目（付与履歴テーブル）
- 付与日 / 付与日数 / 取得済み / 残日数 / 失効日 / 状態

### 表示項目（取得履歴テーブル）
- 取得日 / 取得日数 / 備考

### 制限事項
- 編集・申請機能なし
- 他社員のデータへのアクセス不可（URLに別のIDを指定してもリダイレクト）
