# 有給休暇管理アプリ MVP完成サマリー

## 技術構成

- HTML
- CSS
- JavaScript
- localStorage
- 簡易ログイン認証

## 実装済み機能

### 認証
- 管理者ログイン
- 社員ログイン
- ログアウト
- 未ログインリダイレクト
- 管理者画面のアクセス制御
- 社員マイページのアクセス制御

### 管理者機能
- ダッシュボード
- 社員一覧
- 社員追加
- 社員編集
- 社員詳細
- 有給取得登録
- 退職扱い
- デモデータ初期化

### 社員機能
- マイページ
- 自分の有給残日数表示
- 付与履歴表示
- 取得履歴表示
- 次回付与日表示
- 直近失効日表示

### 有給計算
- 法定付与日数対応
- 2年失効
- 古い付与分から消化
- 半休0.5対応
- 残日数不足チェック

### UI改善（Round10追加）
- ログイン画面にサブタイトル・デモログイン情報カード追加
- 管理者ダッシュボードにクイックアクションカード追加
- ダッシュボードのセクション順を整理（KPI→クイックアクション→失効予定→付与チェック→CSV→データ管理→危険操作）
- 危険操作（デモデータ初期化）を明確に分離
- サイドバーに社員追加リンク追加

### 有給付与チェック（Round9追加）
- 全社員の付与チェック（ダッシュボード）
- 社員別の付与チェック（社員詳細）
- 重複付与防止（同じ employeeId + grantDate は再作成しない）
- 退職者は対象外
- 管理者のみ実行可能

### バックアップ/復元（Round8追加）
- JSONバックアップダウンロード（BOM付きUTF-8ではなくUTF-8）
- バックアップファイルの形式バリデーション
- JSONからのデータ復元（currentUserを除く全データ上書き）
- 復元後にログイン画面へリダイレクト
- 管理者のみ利用可能

### CSV出力（Round7追加）
- 社員一覧CSV（BOM付きUTF-8）
- 有給残日数一覧CSV
- 有給取得履歴CSV
- 社員別詳細CSV（基本情報・サマリー・付与履歴・取得履歴）
- 管理者のみ出力可能

### 比例付与対応（Round19追加）

- 社員データに勤務条件（workType / weeklyWorkDays / weeklyWorkHours / annualWorkDays）を追加
- 通常付与 / 比例付与の自動判定
- 週所定労働日数 1〜4日に対応した比例付与テーブル
- 年間所定労働日数による換算にも対応
- 既存社員はデフォルトで通常付与として扱う

### 有給申請・承認フロー（Round21追加）

- 社員マイページから有給申請（取得希望日・取得日数・申請理由）
- 申請時は残日数を変更しない
- 管理者が有給申請一覧画面で確認・承認・却下
- 承認時に有給取得履歴へ反映・残日数消化
- 却下時は却下理由を保存
- ダッシュボードに未承認申請数KPIを追加

### 有給申請取消（Round24追加）

- 社員が自分の申請中（pending）の有給申請を取消可能
- 取消時は残日数を変更しない
- 取消済み（cancelled）として履歴に残る
- 管理者の申請一覧で取消済みを確認可能
- 有給申請履歴CSVに取消者ID・取消日時を追加

### 有給申請履歴CSV出力（Round23追加）

- 有給申請履歴のCSV出力（管理者専用）
- 申請中・承認済み・却下済みの全申請を出力
- ダッシュボードと有給申請一覧画面の両方に出力ボタンを配置
- ステータスは日本語で出力（申請中 / 承認済み / 却下 / 取消）

### スクリーンショット更新（Round22追加）

- 有給申請フォームのスクリーンショット追加（12_leave_request_form.png）
- 管理者の有給申請一覧スクリーンショット追加（13_leave_requests_admin.png）
- 承認済み申請のスクリーンショット追加（14_leave_request_approved.png）
