# 有給休暇管理アプリ MVP完成サマリー

## 技術構成

- HTML
- CSS
- JavaScript
- localStorage
- 簡易ログイン認証

## 実装済み機能

### 認証
- 管理者ログイン
- 社員ログイン
- ログアウト
- 未ログインリダイレクト
- 管理者画面のアクセス制御
- 社員マイページのアクセス制御

### 管理者機能
- ダッシュボード
- 社員一覧
- 社員追加
- 社員編集
- 社員詳細
- 有給取得登録
- 退職扱い
- デモデータ初期化

### 社員機能
- マイページ
- 自分の有給残日数表示
- 付与履歴表示
- 取得履歴表示
- 次回付与日表示
- 直近失効日表示

### 有給計算
- 法定付与日数対応
- 2年失効
- 古い付与分から消化
- 半休0.5対応
- 残日数不足チェック

### 有給付与チェック（Round9追加）
- 全社員の付与チェック（ダッシュボード）
- 社員別の付与チェック（社員詳細）
- 重複付与防止（同じ employeeId + grantDate は再作成しない）
- 退職者は対象外
- 管理者のみ実行可能

### バックアップ/復元（Round8追加）
- JSONバックアップダウンロード（BOM付きUTF-8ではなくUTF-8）
- バックアップファイルの形式バリデーション
- JSONからのデータ復元（currentUserを除く全データ上書き）
- 復元後にログイン画面へリダイレクト
- 管理者のみ利用可能

### CSV出力（Round7追加）
- 社員一覧CSV（BOM付きUTF-8）
- 有給残日数一覧CSV
- 有給取得履歴CSV
- 社員別詳細CSV（基本情報・サマリー・付与履歴・取得履歴）
- 管理者のみ出力可能
