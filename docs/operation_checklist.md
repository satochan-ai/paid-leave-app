# 有給休暇管理アプリ 動作確認チェックリスト

## 1. ログイン

- [ ] 管理者でログインできる
- [ ] 社員でログインできる
- [ ] 誤ったパスワードでログインできない
- [ ] ログアウトできる

## 2. 権限制御

- [ ] 未ログインで管理者画面に入るとログイン画面に戻る
- [ ] 社員ログインで管理者画面に入るとマイページに戻る
- [ ] 社員は自分のマイページだけ見られる
- [ ] 管理者がマイページにアクセスすると管理者向けメッセージが表示される

## 3. 管理者ダッシュボード

- [ ] 社員数が表示される
- [ ] 在籍社員数が表示される
- [ ] 休職中社員数が表示される
- [ ] 退職者数が表示される
- [ ] 有給残日数合計が表示される
- [ ] デモデータ初期化ボタンが表示される
- [ ] 初期化ボタン押下でconfirmが出る
- [ ] OKするとlocalStorageが初期化される
- [ ] 初期化後、ログイン画面へ戻る
- [ ] 初期ユーザーで再ログインできる

## 4. 社員一覧

- [ ] 社員一覧が表示される
- [ ] 検索ができる
- [ ] ステータス絞り込みができる
- [ ] 詳細画面へ遷移できる
- [ ] 編集画面へ遷移できる
- [ ] 退職扱いができる

## 5. 社員追加

- [ ] 社員を追加できる
- [ ] usersにもログインユーザーが作成される
- [ ] 追加した社員でログインできる
- [ ] 必須項目未入力時にエラーが出る
- [ ] メール重複時にエラーが出る

## 6. 社員編集

- [ ] 社員情報を編集できる
- [ ] users側のメール/パスワード/権限も同期される
- [ ] 編集後の情報でログインできる

## 7. 有給取得登録

- [ ] 1日有給を登録できる
- [ ] 半休0.5日を登録できる
- [ ] 残日数が減る
- [ ] 古い付与分から消化される
- [ ] 残日数不足時にエラーが出る
- [ ] 取得履歴が追加される

## 8. 社員マイページ

- [ ] 自分の氏名が表示される
- [ ] 自分の有給残日数が表示される
- [ ] 自分の付与履歴が表示される
- [ ] 自分の取得履歴が表示される
- [ ] 次回付与日が表示される
- [ ] 直近失効日が表示される

## 9. 表示確認

- [ ] PC表示で大きな崩れがない
- [ ] スマホ表示で大きな崩れがない
- [ ] テーブルが横スクロールできる
- [ ] エラーメッセージが見やすい
- [ ] ログイン画面にプロトタイプ注意書きが表示される
