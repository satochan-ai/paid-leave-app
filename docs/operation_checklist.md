# 有給休暇管理アプリ 動作確認チェックリスト

## 1. ログイン

- [ ] 管理者でログインできる（admin@example.com / admin123）
- [ ] 社員でログインできる（yamada@example.com / password123）
- [ ] 誤ったパスワードでログインできない
- [ ] ログアウトできる
- [ ] ログイン画面にデモ用ログイン情報カードが表示される
- [ ] ログイン画面にプロトタイプ注意書きが表示される

- [ ] 未ログインで管理者画面に入るとログイン画面に戻る
